# Digital Studio - Многостраничный сайт

Современный многостраничный сайт для цифрового агентства.

## Структура сайта

- **index.html** - Главная страница
- **about.html** - О нас
- **services.html** - Услуги
- **portfolio.html** - Портфолио
- **contact.html** - Контакты

## Запуск

### Локальный сервер

```bash
# Python 3
python3 -m http.server 8000

# Или используйте любой другой локальный сервер
# Например, Live Server в VS Code
```

Откройте браузер и перейдите по адресу: `http://localhost:8000`

## Кастомизация

Все цвета и стили можно изменить в файле `css/style.css` через CSS переменные в `:root`.

## Контакты

Для изменения контактных данных отредактируйте соответствующие разделы в HTML файлах.
